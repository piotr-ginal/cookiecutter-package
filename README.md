# Python Package Cookiecutter

A minimal [cookiecutter](https://github.com/cookiecutter/cookiecutter) template for starting modern Python packages.

- Optional CLI with Click
- Pre-configured [ruff](https://github.com/astral-sh/ruff) and [mypy](https://github.com/python/mypy)
- Requirements split for base, dev, typing
- Pre-commit ready

## Usage

```bash
cookiecutter https://github.com/piotr-ginal/cookiecutter-package
```

Generated projects require **Python ≥3.11**.
